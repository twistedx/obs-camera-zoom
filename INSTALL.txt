=============================================
  WEBCAM ZOOM - OBS Filter Plugin
  Installation & Usage Guide
=============================================

WHAT IT DOES
------------
Adds a "Webcam Zoom" filter you can apply to any video source
in OBS. Each source gets its own independent zoom and pan
controls. Uses a GPU shader so it's fast and efficient.


INSTALLATION
------------
1. Open OBS Studio
2. Go to: Tools > Scripts
3. Click the "+" button (bottom left)
4. Browse to this file: webcam-zoom.lua
5. Click "Open" — the filter is now available


ADDING THE FILTER TO A SOURCE
------------------------------
1. Right-click any video source in your scene
2. Select "Filters"
3. Under "Effect Filters", click "+"
4. Choose "Webcam Zoom"
5. Adjust Zoom Level and Pan sliders

You can add this filter to as many sources as you want.
Each one has its own independent settings.


HOTKEYS
-------
Go to: Settings > Hotkeys

Each source with the filter gets its own hotkeys:
  - Webcam Zoom: Zoom In
  - Webcam Zoom: Zoom Out
  - Webcam Zoom: Reset
  - Webcam Zoom: Pan Left / Right / Up / Down


FILTER SETTINGS
---------------
  Zoom Level      — How far to zoom in (1.0 = normal, up to 10x)
  Pan Horizontal  — Left/right position (0.0 = left, 1.0 = right)
  Pan Vertical    — Up/down position (0.0 = top, 1.0 = bottom)
  Zoom Step       — How much each hotkey press zooms
  Pan Step        — How much each hotkey press pans


RESIZE WITHOUT SQUISHING
-------------------------
To prevent squishing when you resize a source to a different
aspect ratio:

1. Right-click the source
2. Transform > Edit Transform
3. Set "Bounding Box Type" to "Scale to outer bounds"

This fills the frame by cropping instead of stretching.


UNINSTALL
---------
1. Remove the "Webcam Zoom" filter from any sources using it
2. Go to: Tools > Scripts
3. Select "webcam-zoom.lua"
4. Click the "-" button to remove it
